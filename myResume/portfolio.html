Flappy Bird - It is the very famous Flappy Bird game made using Processing which is an open-source graphical library and integrated development environment (IDE) and it uses Java as the language. As I have coded the game on my phone, I have made it to be playable on mobile devices only by keeping the orientation as portrait.

Snake Game - A Snake Game built using P5.js which is a library of JavaScript. The game is built by keeping the orientation as portrait to be playable on a mobile phone.

Covid HERO - This was a submission project made for The Open Hacks hackathon, organised by MLH. The chatbot uses Dialogflow and the voice part uses Google Assistant API The goal of the project was to create a chatbot that can give the users relevant information about COVID-19 during the start of pandemic.

Dog Breed Classifier - In this project, I have built a Convolutional Neural Network (CNN) using Tensorflow and Keras that can classify the breed of dog from any user-supplied image. If the image is of a human and not a dog, the algorithm will provide an estimate of the dog breed that is most resembling.

TV Script Generator - In this project, I have constructed a Recurrent Neural Network (RNN) model using Tensorflow and Keras to generate it's own Seinfeld TV scripts by using a Seinfeld dataset of scripts from 9 seasons. The Neural Network generated a new, "fake" TV script for a hypothetical Season 10 of the show.

Azure Telemetry Service - This project is based on Microsoft Azure's feature: IoT Hub. This project is all about sending telemetry messages to devices like raspberry pi and get back their topological info through a VM (Virtual Machine).
